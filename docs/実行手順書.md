# Claude Skills Platform 実行手順書

## このプロジェクトについて

「AI Skills Library」は、Eudaimoniaが提供するWebサイトです。
ビジネスで使えるAIスキル（プロンプト集）を一覧表示し、ワンクリックでダウンロードやコピーができます。

**技術スタック**: Next.js + React + TypeScript + Tailwind CSS

---

## 前提条件

以下がパソコンにインストールされている必要があります。

| 必要なもの | 確認コマンド | 推奨バージョン |
|-----------|-------------|--------------|
| Node.js | `node -v` | v20以上 |
| npm | `npm -v` | v10以上 |

> **Node.jsが入っていない場合**
> https://nodejs.org/ から「LTS」版をダウンロードしてインストールしてください。
> npmはNode.jsと一緒に自動でインストールされます。

## 手順3: パッケージをインストールする

```bash
npm install
```

**何が起こるか:**
- プロジェクトに必要なライブラリ（部品）が自動でダウンロードされます
- `node_modules` フォルダが作成（または更新）されます
- 初回は1〜3分程度かかることがあります

**成功の目安:**
`added XX packages` のようなメッセージが出ればOKです。
`npm warn` は警告なので無視してOKです。エラー（`npm ERR!`）が出た場合のみ対処が必要です。

> **既に `node_modules` フォルダが存在する場合**
> この手順はスキップしても動く可能性がありますが、念のため実行を推奨します。

---

## 手順4: 開発サーバーを起動する

```bash
npm run dev
```

**何が起こるか:**
- ローカル（自分のPC内）でWebサーバーが起動します
- インターネットに公開されるわけではありません（自分だけが見られます）

**成功の目安:**
以下のようなメッセージが表示されます。

```
  ▲ Next.js 16.x.x
  - Local:   http://localhost:3000
  - Network: http://192.168.x.x:3000

 ✓ Ready in X.Xs
```

## トラブルシューティング

### Q: `npm install` でエラーが出る

```bash
# node_modulesを削除して再インストール
rm -rf node_modules package-lock.json
npm install
```

### Q: `npm run dev` でポート3000が使われていると言われる

別のプロセスが3000番ポートを使用中です。

```bash
# ポートを変えて起動
npm run dev -- --port 3001
```

→ http://localhost:3001 で確認できます。

### Q: ページが真っ白に表示される

ブラウザの開発者ツール（`Cmd + Option + I`）を開き、Consoleタブでエラーを確認してください。

### Q: Node.jsのバージョンが古いと言われる

```bash
# 現在のバージョン確認
node -v

# v20未満の場合はNode.jsを再インストール
# https://nodejs.org/ から最新LTS版をダウンロード
```

## スキルを追加・編集したいとき

`src/data/skills.ts` ファイルを編集します。

各スキルは以下の形式で定義されています:

```typescript
{
    id: 'skill-id',           // 英数字のID（ユニーク）
    name: 'スキル名',          // 表示名
    description: '短い説明',    // カード上の説明文
    longDescription: '詳細説明', // 詳細ページの説明
    category: 'カテゴリ名',     // フィルター用カテゴリ
    iconName: 'IconName',      // lucide-reactのアイコン名
    author: 'Eudaimonia',      // 作者名
    version: '1.0.0',          // バージョン
    content: `プロンプト本文`,   // 実際のスキル内容
    tags: ['タグ1', 'タグ2'],   // 検索用タグ
    timeSaved: '30分/回',       // 節約時間の目安
    difficulty: 'かんたん'       // 難易度（かんたん/ふつう/しっかり）
}
```

ファイルを保存すると、開発サーバー起動中であれば自動でページが更新されます。
